#  Lab 01: API Security Fundamentals

##  Overview
This lab introduces the core concepts of API security, focusing on how APIs function and why they are a critical attack surface in modern applications.

##  Objective
To understand API architecture, common security risks, and how vulnerabilities arise due to improper testing and exposure.

##  Tools Used
- Postman
- Docker
- VS Code

## Approach
- Set up local API environment
- Explored API endpoints using Postman
- Reviewed API responses and behaviors
- Studied OWASP API Top 10 risks

##  Findings
- APIs often expose sensitive endpoints
- Security is frequently overlooked in favor of functionality
- Improper testing can lead to vulnerable APIs

##  Mitigation
- Implement security testing alongside functional testing
- Follow OWASP API Security Top 10 guidelines
- Restrict unnecessary endpoint exposure
