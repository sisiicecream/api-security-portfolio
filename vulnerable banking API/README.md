#  Lab 04: API Security Testing

##  Overview
This lab focuses on identifying API vulnerabilities through testing and response analysis.

##  Objective
To understand how improper validation and error handling expose sensitive information.

##  Tools Used
- Postman
- Browser DevTools

## Approach
- Sent crafted API requests
- Observed error messages and responses
- Tested input validation mechanisms

##  Findings
- Error messages revealed internal system details
- Improper validation allowed unsafe inputs
- API responses exposed sensitive information

##  Mitigation
- Implement proper input validation
- Use generic error messages
- Avoid exposing internal system details
